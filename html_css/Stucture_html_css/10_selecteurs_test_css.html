Proprietés CSS utilisées dans cette page

Decrivez leurs usages et leurs valeurs possibles

  1. Backgroun: couleur de fond de l'élément
  2. Color: couleur de la police
  3. Float: effet "flottant"
  4. Font-Family: police
  5. Font-size: taille de la police
  6. Height: hauteur de l'élément
  7. Left: position à partir de la gauche de la fenêtre
  8. Line-height: hauteur de l'interligne
  9. Margin: marges exterieures>
  10. Padding: marges interieurs
  11. Position: position de l'élément dans la page (on a utilisé que la position fixed pour l'instant)
  12. Text-align: alignement du texte
  13. Text-decoration: changement des valeurs héritées (ici le souligné qui est standard pour les liens sur tous les navigateurs est supprimé)
  14. Top: position à partir du haut de la fenêtre
  15. Transition: temps de transition lors des animations (ici sur les liens de la barre de menu)
  16. Width: largeur de l'élément par rapport a la taille de la fenêtre si on indique une valeur relative (%, em....)
