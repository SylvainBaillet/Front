Les boucles

Les instructions for et while qui sont deux types de boucles les plus utilisées en JS, servent a repeter plusieurs lignes de code.

La boucle for

For prend 3 parametres encadrés dans des parentheses, suivi entre accolades des instructions a executer autant de fois que la boucle l'indique.les parametres sont

  • Initialisation: ici on declare la variable qui servira de compteur
  • condition: Tant que cette condition est vraiela boulce s'executera
  • Expression finale; expression qui est executée a chaque fin de boucle.
for (initialisation; condition; incrementation){ instruction 1; instruction 2; }
