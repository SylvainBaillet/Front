Les instrutions

Un script js est un fichier contenant des instructions, placées les unes a la suite des autres.Chacune de ces instructions peut etre vue comme un ordre doné au moteur javaScript, une action que l'on souhaite réaliser a la maniere d'une recette de cuisine.

  • Faire fondre le beurre
  • Melanger le beurre et la farine
  • Verser le chocolat petit a petit et...

Les instructions se suivent et sont effectuées les unes a la suite des autres, du haut vers le bas(comme en CSS). Le browser parse (analyse) dans l'ordre du flux

Chaque instruction DOIT se terminer par un point virgule, afin d'indiquer au moteur javaScript que cette instruction est terminée.

Les fonctions

Le fonctions permettent de regrouper une serie d'instructions pour pouvoir les reutiliser. Quand on a besoin du code d'une serie d'instruction.Plutot que de ré ecrire a chaque fois toutes les instructions, on appellera une fonction par son nom (faireUnGateau). Cette fonction contient l'ensemble des instructions a executer.

Par exemple la fonction alert est une fonction native integrée au navigateur, qui contient en gros les instructions suivantes:

  • Creer le cadre de la boite de dialogue
  • Positionner le cadre dans la fenetre
  • Creer un backgroud semi transparent sur toute la page
  • Placer un bouton avec le texte ok et un evenement click
  • Creer une zone de texte qui affichera de l'art a presenterau visiteur

Nous on a juste besoin d'utiliser la fonction alert() en lui donnant un argument, ex ("coucou") La fonction executera toutes les instructions a chaque fois
On peut utiliser cette fonction alert quand on veut en la declenchant avec : alert("quelquechose") Ce quelquechose est un ARGUMENT attendu par la fonction.

var maVariable = document.querySelector('h2'); alert('coucou');

Les fonctions querySelector et alert sont disponibles dans le navigateur, ce sont des methodes

si vous voyez quelquechose qui ressemble a une variable et qui est suivie de parenthese, c'est une fonction.

Les fonctions prennent souvent des arguments, des bouts de données dont elles ont besoin pour faire leur travail? Les arguments sont placés ENTRE les parentheses, separées par des virgules si il y'en a plusieurs

Par exemple la fonction alert , fait apparaitre une popup mais vous devez donner une chaine comme argument pour indiquer a la fonction ce que vous souhaitez ecrire dans la fenetre.
