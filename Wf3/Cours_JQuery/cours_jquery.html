Selectionner en jQuery

Le coeur de jQuery se trouve dans la selection d'elements pour ensuite executer des methodes dessus

$(selecteur) accepte en argument tout selecteur CSS, jQuery nous renvoie ensuite une collection d'un ou plusieurs éléments DOM enveloppés dans un objet jQuery.

$('div') selectionne tous les divs $('.titre') selectionne tous les éléments qui ont la class='titre'

Cette enveloppe jQuery est celle qui donne acces a toutes les fonctionnalités de jQuery

  • Rajouter des classes
  • Modifier le contenu HTML
  • Afficher ou cacher des éléments
  • Les animer etc...

Tous les éléments selectionnés en javaScript pur (document.getElementById, querySelector etc...) sont des elements DOM et tous ceux selectionnés avec $ ('') sont des elements DOM dans une enveloppe jQuery

jQuery dispose de plusieursmethodes pour parcourir l'arbre du document (le DOM)

  • children() parcourt les enfants
  • find() parcourt les descendants
  • next() renvoie le voisin suivant
  • prev() renvoie le voisin precedent
  • parent() renvoie le parent
  • eq() renvoie parmi une liste d'elements, celui qui se trouve a l'index fourni
