Valeurs

Observez le programme valeurs.js et tentez de prevoir les valeurs finales de chaque variable.

var a = 2;

a = a - 1;

a++;

var b = 8;

b += 2;

a++;

var c = a + b * b;

var d = a * b + b;

var e = a * (b + b);

var f = a * b / a;

var g = b / a * a;

Verifiez vos previsions en creant ce programme puis en le deboguant pour observer les valeurs des variables.
